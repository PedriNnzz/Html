Como podemos utilizar o HTML, CSS, JS

HTML é uma linguagem de programaçao de criaçao de sites, se tiver alguma duvida sobre a linguagem basta clicar em Html.

CSS é outra linguagem que se conecta com HTML, pois fica responsavel por dar vida no site(desing), qualquer duvida basta clicar em Css.

Por ultimo tesmos o javascript, que é responsavel por funçoes interativas e dinamicas do site, qualquer duvida basta clicar em JavaScript.
